## 🚀 Overview

Emby-In-One is a Windows app that lets you use one Emby entry point for multiple upstream Emby servers. It gathers media from several servers and presents them through one unified access point.

If you use more than one Emby server, this tool helps you keep access simple. You can open one client, sign in once, and reach the merged library through a single path.

## ✨ What It Does

- Combines multiple Emby server sources into one view
- Gives you one place to access your media library
- Works with standard Emby clients
- Fits VPS-based setups
- Keeps the user side simple

## 🖥️ Who It Is For

Use Emby-In-One if you:

- Run more than one Emby server
- Want one clean entry for all media
- Use Emby on Windows, TV boxes, or mobile clients
- Need a front layer in front of several upstream servers
- Want a setup that feels like one server from the client side

## ⚙️ First-Time Use

After you start the app, set up your upstream Emby servers one by one.

Typical setup steps:

1. Add the first Emby server address.
2. Add the login details or access token for that server.
3. Add the next server.
4. Repeat until all servers are listed.
5. Save the settings.
6. Open your Emby client and connect to the unified entry point.

If the app uses a local web address or proxy address, copy that address into your Emby client. If it asks for a port, keep the default unless your setup needs a different one.
